# 세 번째 - 개발을 위한 기본 버전


## 구성
- react + babel 을 이용한 페이지 코딩
- webpack을 이용한 bundling

## 설명

- `./src/` 의 `index.html` 파일을 `./dist/` 로 복사
- `./src/` 의 소스(js)들을 `./dist/` 의 `app.js` 으로 합침. 

## 순서


### WEBPACK 설치

`$ npm install webpack -g`
> webpack command 실행을 위해 webpack을 global로 설치.

### 로컬 실행

- ` npm install ` 으로 node 모듈들 설치
- `webpack` 실행
